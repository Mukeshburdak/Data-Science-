# 📈 Simple Linear Regression – Salary Prediction

This project demonstrates the implementation of Simple Linear Regression using Python to predict salaries based on years of experience. It is a beginner-friendly machine learning project that explains how regression models work using a real dataset.

The model learns the relationship between Years of Experience and Salary and predicts salary values for new inputs.

# 📂 Project Structure
Simple-Linear-Regression/

├── Simple_Linear_Regression.ipynb   # Jupyter Notebook with full implementation

├── Salary_dataset.csv               # Dataset used for training and testing

└── Sol.md                        # Project documentation
# 📊 Dataset

The dataset contains two columns:

Column	Description
YearsExperience	Number of years of professional experience
Salary	Salary corresponding to the experience

This dataset is commonly used to demonstrate linear regression concepts.
